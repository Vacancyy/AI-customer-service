# AI客服影子测试接入指南

## 一、接入方案概述

**影子测试**：AI在后台生成回答并记录，但**不展示给用户**，用户仍由人工客服接待。

目的：
- 收集真实客户问题
- 对比AI回答与人工回答
- 验证AI准确度后再正式上线

---

## 二、架构图

```
合力客服系统                          影子测试服务器
    │                                    │
    │ ①客户发消息                         │
    │─────────POST /shadow/question──────►│
    │                                    │ AI生成回答
    │                                    │ 记录问题+AI回答
    │                                    │
    │ ②人工客服回答                       │
    │─────────POST /shadow/answer────────►│
    │                                    │ 补充人工回答
    │                                    │
    │                                    │ ③定期生成对比报告
```

## 四、合力客服系统接入代码

### 4.1 接收客户消息时调用

```javascript
// 合力客服系统 - 客户发消息时触发
function onCustomerMessage(sessionId, visitorId, question) {
    // 调用影子测试API
    fetch('http://192.168.10.170:8001/shadow/question', {
        method: 'POST',
        headers: {'Content-Type': 'application/json'},
        body: JSON.stringify({
            session_id: sessionId,
            visitor_id: visitorId,
            question: question,
            question_time: new Date().toISOString()
        })
    }).then(response => response.json())
    .then(data => {
        // 影子测试：AI回答已记录，但不展示
        console.log('影子测试记录:', data.log_id);
    });
}
```

### 4.2 客服回答时调用

```javascript
// 合力客服系统 - 客服发送回答时触发
function onAgentAnswer(sessionId, visitorId, agentId, agentName, answer) {
    // 调用影子测试API
    fetch('http://192.168.10.170:8001/shadow/answer', {
        method: 'POST',
        headers: {'Content-Type': 'application/json'},
        body: JSON.stringify({
            session_id: sessionId,
            visitor_id: visitorId,
            agent_id: agentId,
            agent_name: agentName,
            answer: answer,
            answer_time: new Date().toISOString()
        })
    }).then(response => response.json())
    .then(data => {
        console.log('人工回答已记录:', data.log_id);
    });
}
```

## 五、API接口详情

### 5.1 POST /shadow/question

**用途**：接收客户问题，后台生成AI回答并记录

**请求参数**：
```json
{
    "session_id": "会话ID",
    "visitor_id": "客户ID",
    "question": "客户问题内容",
    "question_time": "2026-05-14 10:30:00"  // 可选
}
```

**返回示例**：
```json
{
    "success": true,
    "log_id": 123,
    "ai_generated": true,
    "ai_category": "high_confidence",
    "ai_match_score": 0.85,
    "process_time": 1.5,
    "message": "影子测试：AI回答已生成并记录，未展示给用户"
}
```

**注意**：不返回AI回答内容！这是影子测试的核心特性。

---

### 5.2 POST /shadow/answer

**用途**：补充人工客服回答

**请求参数**：
```json
{
    "session_id": "会话ID",
    "visitor_id": "客户ID",
    "agent_id": "客服ID",
    "agent_name": "南京在线1",
    "answer": "您好，理赔门槛是...",
    "answer_time": "2026-05-14 10:30:05"  // 可选
}
```

**返回示例**：
```json
{
    "success": true,
    "log_id": 123,
    "response_seconds": 5.0,
    "message": "人工回答已记录，可进行对比分析"
}
```

---

### 5.3 GET /shadow/stats

**用途**：查看影子测试统计

**返回示例**：
```json
{
    "total_questions": 500,
    "with_agent_answer": 480,
    "avg_agent_response_time": 30.5,
    "categories": [
        {"category": "high_confidence", "count": 200, "avg_score": 0.82},
        {"category": "medium_confidence", "count": 250, "avg_score": 0.60},
        {"category": "low_confidence", "count": 30, "avg_score": 0.40}
    ]
}
```

---

### 5.4 GET /shadow/export

**用途**：导出对比报告

**返回**：
```json
{
    "success": true,
    "report_path": "05_analyze/reports/影子测试对比报告.md",
    "summary": {
        "total_records": 480,
        "ai_answered_ratio": 93.8,
        "avg_ai_time": 1.5,
        "avg_agent_time": 30.5
    }
}
```

## 六、接入检查清单

### 合力客服系统需要：

| 项目 | 说明 |
|------|------|
| ① 识别客户消息 | 在客户发消息时调用 `/shadow/question` |
| ② 识别客服回答 | 在客服发送回答时调用 `/shadow/answer` |
| ③传递会话信息 | session_id、visitor_id、agent_id、agent_name |

### 影子测试服务器需要：

| 项目 | 说明 |
|------|------|
| ① 环境变量配置 | DASHSCOPE_API_KEY、数据库配置 |
| ② 启动API服务 | `python scripts/shadow_api.py` |
| ③ 开放端口 | 8001端口允许合力系统访问 |

## 七、试运行流程

```
第1天：启动影子测试API，合力系统接入调用
第2-7天：收集数据，每天查看 /shadow/stats
第7天：调用 /shadow/export 生成对比报告
第8天：根据报告决定是否正式上线AI
```

---

## 八、对比报告内容

报告包含：
- AI vs 人工响应时间对比
- AI处理能力分析（高/中/低置信度分布）
- 典型案例对比（AI回答 vs 人工回答）
- 准确度评估
